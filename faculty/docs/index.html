Pages Layout

The main area is filled with pages, you can add as many pages as you want.

Each page required to have an id so it can be referenced within the menus and a class="page" whitch indicates that this div is a page.

For the Home page of the site we set two extra parameters which are class of class="home" and data-pos="home" so that when you click on the homepage at the menu, this one would act different.

Notice In order to sync the menu with the pages, the href attribute of the menu item should properly point to the id of the page. For example, if your page has an id of research like id="research", the anchor at the menu should refer to href="#research".

    Filters

    Did you noticed the class of cpaper at the first line? Well, that is the key to filtering. Each publication falls in a group so we can filter them by that group. You can add as many groups as you want, though.

    and again, there is a cpaper there as a value of the option. When ever a user clicks on the select element, the mixitup jquery plugin will simply hide elements except those has class of the value selected. So be carefull about this.
